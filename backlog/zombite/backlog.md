# Backlog: zombite

> Generated by `aitri plan`.
> Spec-driven rule: every story MUST reference one or more Functional Rules (FR-*) and, when applicable, Acceptance Criteria (AC-*).

## Epics
- EP-1: Core loop de gameplay corto y divertido
  - Notes: entrada inmediata a partida, disparo responsivo y cierre claro de cada nivel.
  - Trace: FR-1, FR-2
- EP-2: Progresion y distribucion embebible segura
  - Notes: zombies alfa, configuracion de widget y validaciones de seguridad.
  - Trace: FR-3, FR-4

## User Stories

### US-1
- As a jugador casual, I want iniciar una partida y empezar a disparar en segundos, so that puedo distraerme rapidamente durante descansos cortos.
- Trace: FR-1, AC-1
- Acceptance Criteria:
  - Given el juego cargado, when presiono iniciar, then entro al nivel 1 con controles activos y HUD visible.
  - Given una sesion corta, when completo o pierdo un nivel, then recibo feedback claro y opcion inmediata de continuar o reiniciar.

### US-2
- As a jugador casual, I want progresion de dificultad por niveles, so that el juego se mantenga entretenido sin volverse frustrante.
- Trace: FR-2, AC-2
- Acceptance Criteria:
  - Given la partida en nivel bajo, when avanzo niveles, then aumentan enemigos y exigencia de precision de forma gradual.
  - Given nivel 10, when lo juego con habilidad media, then sigue siendo retador pero alcanzable.

### US-3
- As a jugador casual, I want enfrentar zombies alfa de forma ocasional, so that haya picos de emocion y variedad.
- Trace: FR-3, AC-3
- Acceptance Criteria:
  - Given un nivel con evento alfa, when aparece el enemigo especial, then se distingue visualmente y requiere mas impactos.
  - Given derrota del alfa, when termina el encuentro, then el progreso del nivel continua sin errores.

### US-4
- As a sitio anfitrion, I want incrustar el juego como iframe/widget con configuracion segura, so that pueda reutilizarlo sin comprometer estabilidad.
- Trace: FR-4, AC-4
- Acceptance Criteria:
  - Given configuracion valida del widget, when carga el juego, then aplica idioma/volumen/dificultad inicial correctamente.
  - Given configuracion malformada o no permitida, when se procesa, then se ignora y se usan defaults seguros.
